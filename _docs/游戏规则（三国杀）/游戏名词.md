+ 平台是基于三国杀标准包五人身份场模式的，并且跳过选择武将的环节，每局游戏随机分配武将、身份和座位次序。该模式包含4个身份共5个角色，其中反贼数为2。
    + **主公**：优先出牌，身份开局亮出，全场可知。
    + **忠臣**：身份开局隐藏，阵亡后身份公布。
    + **反贼**：身份开局隐藏，阵亡后身份公布。
    + **内奸**：身份开局隐藏，阵亡后身份公布。
